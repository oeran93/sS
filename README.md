# slideShow

JQuery plugin to build slideShows.

###Description
This simple plugin allows to easily set up a div with sliding background images.

###Example
####File.js
```
$('#slideShow').sS(
					{
						path: './images/',
						images: [
							'1.jpg',
							'2.jpg',
							'3.jpg',
							'4.jpg'
						]
					}
);
```
####File.html
```
<div id='slideShow'></div>
```
